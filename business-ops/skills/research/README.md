# Research

Research skill slot for the `business-ops` plugin.

## Current status

This README is intentional, but the underlying [`SKILL.md`](./SKILL.md) is still effectively empty. The folder already contains `examples/`, `hooks/`, `references/`, `scripts/`, and `templates/`, so the package shape exists even though the core instructions still need to be written.

## Install

```bash
npx -y skills add ./business-ops/skills/research
mkdir -p ~/.codex/skills
cp -R business-ops/skills/research ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/business-ops/skills/research
```

Until the main skill instructions are completed, treat this as a reserved slot rather than a finished skill.
