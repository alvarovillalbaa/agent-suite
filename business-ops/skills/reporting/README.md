# Reporting

Turn raw metrics, notes, and developments into decision-ready weekly, monthly, board, or operating reports.

## Use this for

- weekly updates and monthly operating reviews
- board and investor updates
- KPI summaries that need narrative, not just numbers
- executive reporting with trends, drivers, risks, asks, and next actions

## Install

```bash
npx -y skills add ./business-ops/skills/reporting
mkdir -p ~/.codex/skills
cp -R business-ops/skills/reporting ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/business-ops/skills/reporting
```

## Notes

This skill is strongest when the input is messy but the output needs to be concise and executive-readable. See [`SKILL.md`](./SKILL.md) for the reporting structure and format modes.
