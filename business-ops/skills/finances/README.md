# Finances

Unified finance skill covering reconciliation, financial modeling, and fundraising support.

## Use this for

- reconciling statements, ledgers, receipts, and audit trails
- building or reviewing P&L, cash flow, scenario, or cap-table models
- turning decks, models, CRM notes, and investor feedback into fundraising materials or diagnostics

## Install

```bash
npx -y skills add ./business-ops/skills/finances
mkdir -p ~/.codex/skills
cp -R business-ops/skills/finances ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/business-ops/skills/finances
```

## What is bundled

- `examples/`
- `hooks/`
- `references/`
- `scripts/`
- `templates/`

The authoritative workflow lives in [`SKILL.md`](./SKILL.md).
