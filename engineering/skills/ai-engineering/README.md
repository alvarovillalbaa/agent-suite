# AI Engineering

AI and data engineering skill for ML systems, pipelines, prompt workflows, evaluation, and production AI architecture.

## Use this for

- data pipelines, ETL/ELT, streaming, and data quality work
- prompt design, prompt optimization, RAG, and agent workflows
- classical ML evaluation and experimentation
- computer vision systems and model-serving decisions

## Install

```bash
npx -y skills add ./engineering/skills/ai-engineering
mkdir -p ~/.codex/skills
cp -R engineering/skills/ai-engineering ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/engineering/skills/ai-engineering
```
