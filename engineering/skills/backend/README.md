# Backend

Backend engineering skill for APIs, services, schemas, persistence, and server-side operational work.

## What it owns

- API design and review rules
- service-layer, queue, auth, and persistence patterns
- schema design, normalization, index strategy, and database optimization
- backend-specific helper scripts for API linting, load testing, scaffolding, and migrations

## Relationship to `agentic-development`

Use [`../agentic-development/SKILL.md`](../agentic-development/SKILL.md) for repo orientation, planning, execution-mode selection, and coordinated full-stack work. Use this skill for the backend slice itself.

## Install

```bash
npx -y skills add ./engineering/skills/backend
mkdir -p ~/.codex/skills
cp -R engineering/skills/backend ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/engineering/skills/backend
```
