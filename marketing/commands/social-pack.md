Use skill: **content-syndication** — `skills/content-syndication/SKILL.md`. Optionally use **linkedin-articles** or **x-articles** if one hero format needs deeper treatment.

1. **Confirm the source asset** – Ask for the canonical article, draft, memo, or idea that should anchor the pack.
2. **Confirm the channels** – Default to LinkedIn and X unless the user asks for a wider pack.
3. **Generate the pack** – Use `content-syndication` to create channel-native variants with differentiated hooks and framing.
4. **Deepen one format if needed** – If the user wants a hero X or LinkedIn long-form asset, route through `x-articles` or `linkedin-articles`.
5. **Deliver** – Output the per-channel drafts and a suggested publish order.
