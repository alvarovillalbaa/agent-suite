# SEO Writing Checklist

Full on-page SEO checklist and content template for use with the Blog mode's SEO Integration workflow.

---

## On-Page SEO Checklist

**Keyword Placement**:
- [ ] Primary keyword in title (preferably at start)
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword in conclusion
- [ ] Primary keyword in meta description
- [ ] Secondary keywords in H2s/H3s
- [ ] Related terms throughout body (LSI/semantic variants)

**Content Quality**:
- [ ] Comprehensive coverage of topic
- [ ] Original insights or data
- [ ] Actionable takeaways
- [ ] Examples and illustrations
- [ ] Expert quotes or citations (for E-E-A-T)
- [ ] ≥1 external citation per 500 words
- [ ] Every major claim backed by evidence

**Readability**:
- [ ] Paragraphs of 3–5 sentences
- [ ] Varied sentence length
- [ ] Bullet points and lists for scannable content
- [ ] Bold key phrases (sparingly, not every other line)
- [ ] Table of contents for content over 1,500 words

**Technical Elements**:
- [ ] Internal links to relevant pages (2–5)
- [ ] External links to authoritative sources (2–3)
- [ ] Image alt text with relevant keywords
- [ ] URL slug includes primary keyword

**Metadata**:
- [ ] SEO title: ≤60 chars, includes primary keyword
- [ ] Meta description: 150–160 chars, includes keyword + CTA
- [ ] URL slug: clean, hyphenated, keyword-inclusive

## Featured Snippet Optimization Patterns

**For Definition Snippets**:
Place immediately under the relevant H2:
`"[Term] is [clear, concise definition in 40–60 words]"`

**For List Snippets**:
Create clear numbered or bulleted lists under H2s. Keep list items parallel in structure.

**For Table Snippets**:
Use comparison tables with clear, descriptive headers. Include at least 3 rows.

**For How-To Snippets**:
Number each step explicitly — "Step 1: …", "Step 2: …". Each step should be a single actionable instruction.

**For FAQ Snippets**:
Format questions as H3s. Answers: 40–60 words, starts with a direct statement that could stand alone.

## SEO Validation Checkpoints

### Input Validation
- [ ] Primary keyword confirmed and matches search intent
- [ ] Target word count specified (minimum 800 words for substantive content)
- [ ] Content type and audience clearly defined
- [ ] Competitor URLs reviewed or target SERP features identified

### Output Validation
- [ ] All sections from outline covered completely
- [ ] Keyword density feels natural (1–2% guideline — prioritize semantic relevance over exact count)
- [ ] Internal links included (2–5 relevant links)
- [ ] External links included (2–3 authoritative sources)
- [ ] FAQ section present with at least 3 questions
- [ ] Metadata complete: title, meta description, slug
