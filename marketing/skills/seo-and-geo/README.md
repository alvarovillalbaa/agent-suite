# SEO And GEO

Search optimization skill spanning SEO, GEO, and AEO.

## Use this for

- on-page SEO audits and full technical SEO checks
- GEO work for AI visibility and AI assistant citations
- AEO work for snippets, AI Overviews, and direct-answer surfaces
- competitor benchmarking, content gaps, and keyword opportunities
- structured-data, Core Web Vitals, crawlability, i18n, and link-building workflows

## Install

```bash
npx -y skills add ./marketing/skills/seo-and-geo
mkdir -p ~/.codex/skills
cp -R marketing/skills/seo-and-geo ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/marketing/skills/seo-and-geo
```
