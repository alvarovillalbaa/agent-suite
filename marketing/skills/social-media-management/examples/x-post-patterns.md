# X Post Patterns — High-Performing Examples

Annotated examples showing structure, why they work, and how to apply each pattern.

---

## Pattern 1: The Specific-Number Listicle Hook

**Format:** Single post or thread teaser

```
I've reviewed 300+ landing pages.

The ones that convert share 4 things:

• A headline that names the outcome (not the product)
• Social proof in the first scroll
• One CTA, repeated 3 times
• A price anchored below perceived value

Most pages get 0 of these right.
```

**Why it works:**
- "300+" establishes credibility without claiming perfection
- Each bullet is a specific, actionable principle — not a generic tip
- Last line creates contrast ("most pages") and makes the reader feel they now have an edge
- No CTA needed — saves organically

**Apply when:** Sharing a framework derived from real experience or research

## Pattern 2: The Bold Contrast

**Format:** Single post, high engagement

```
Everyone tells you to post consistently.

No one tells you that posting bad content consistently just accelerates your audience's decision to unfollow.

Quality → Consistency. Not the other way.
```

**Why it works:**
- Opens with a widely accepted belief, immediately challenges it
- Second line is the real insight — it inverts the standard advice with a specific mechanism
- Third line lands the principle cleanly — quotable, screenshot-able
- Generates replies from both sides

**Apply when:** You have a contrarian take that has real evidence behind it

## Pattern 3: The Before/After Story

**Format:** Thread or long single post

```
6 months ago: 2,000 followers, posting every day, zero engagement.

Today: 47,000 followers, posting 3x a week, consistently 100k+ impressions.

The one change: I stopped writing for an audience I wanted and started writing for the audience I had.

Thread on what that actually meant in practice:
```

**Why it works:**
- Opens with the end state — curiosity created immediately
- Specific numbers (not "grew a lot") — credibility
- The "one change" creates a clear promise for the thread
- "In practice" signals specificity is coming

**Apply when:** You have a genuine transformation or case study with real data

## Pattern 4: The Resource Save

**Format:** Single post, bookmarked often

```
The 10 X accounts every founder building in public should follow:

1. @handle — [one-line description of what makes them worth following]
2. @handle — [...]
3. @handle — [...]
(... continue)

Save this. Revisit it when you need perspective.
```

**Why it works:**
- Curated lists are inherently bookmark-worthy
- One-line descriptions justify each pick — not a vanity list
- "Save this" CTA matches the format (people were already going to save it)

**Apply when:** You can genuinely curate a list where each item has a specific reason for inclusion

## Pattern 5: The Data Counterintuitive

**Format:** Single post or thread teaser

```
We analyzed 1,000 viral posts from the last 90 days.

The posts with the most engagement had fewer words — not more.

Median length of a 1M+ impression post: 47 words.

Write less. Say more.
```

**Why it works:**
- Opens with a credibility signal (data source, sample size, timeframe)
- The finding is counterintuitive — violates what people think they know
- The specific number (47 words) is precise enough to feel real
- Last line is quotable and actionable

**Apply when:** You have access to data (even rough) that challenges a common assumption

## Pattern 6: The Step-by-Step Thread

**Format:** Thread

```
Tweet 1: "How to go from idea to published blog post in 2 hours. A thread:"

Tweet 2: "STEP 1 — DON'T START WITH AN OUTLINE
Start with your conclusion.
Write the last paragraph first.
It forces you to know what you're actually arguing.
Most drafts fail because the writer never decided what they think."

Tweet 3: "STEP 2 — WRITE THE INTRO LAST
Your intro is a promise.
You can only write a good promise when you know what you're delivering.
Write it once the draft exists."

(... continue per step)

Final tweet: "Follow @handle for one writing framework a week.
I publish every Tuesday."
```

**Why it works:**
- Capitalized step headers create clear navigation
- Each step explains WHY, not just WHAT to do
- Banger lines end each tweet ("Most drafts fail because...")
- Final tweet CTA is specific (day, cadence)

**Apply when:** Teaching a process where sequence matters and each step has a non-obvious reason

## Anti-Patterns to Avoid

| Pattern | Why it fails |
|---------|--------------|
| "Excited to announce..." | Self-centered opener — reader doesn't care yet |
| "Great question" as opener | Filler that kills momentum |
| "In conclusion..." | Signals the end too early |
| "Thread 🧵" as the only hook | No reason to click through |
| 10-item list with no explanation | Looks lazy; gets skipped |
| "Like and repost if you agree" | Feels manipulative; suppressed algorithmically |
| Generic inspirational quote | Zero differentiation |
