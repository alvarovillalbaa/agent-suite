# X Post Formats

A library of high-performing post and thread structures for X.

---

## Single Posts

### Hook + Evidence + CTA
Best for: standalone insights, learnings, or frameworks

```
[Bold claim or surprising insight]

[One specific piece of evidence, example, or stat]

[What to do with this — follow / save / reply]
```

### Numbered Teaser
Best for: driving thread reads, listicle hooks

```
[Number] [things/ways/lessons] about [topic] after [specific context]:

• [Item 1 — one line]
• [Item 2 — one line]
• [Item 3 — one line]

[Thread below / Full breakdown:]
```

### Hot Take
Best for: engagement, debate, establishing a point of view

```
Unpopular opinion: [Contrarian stance in one line]

[The strongest evidence or reason behind it]

[What this means for the reader]
```

### Question Post
Best for: starting conversations, community building

```
[Specific, thought-provoking question about a shared experience?]

My take: [Your concrete answer in 1–2 sentences]

What's yours?
```

### Resource / Save Post
Best for: bookmark-worthy utility content

```
[Topic]: [specific benefit]

• [Actionable item 1]
• [Actionable item 2]
• [Actionable item 3]
• [Actionable item 4]
• [Actionable item 5]

Bookmark this.
```

### Milestone / Learning Post
Best for: building credibility, transparency, audience trust

```
[Specific milestone reached].

Here's the one thing I'd do differently:

[Concrete, specific lesson — not a generic platitude]

[Why it matters to the reader]
```

## Thread Structures

### How-To Thread
Best for: teaching a process, establishing expertise

```
Tweet 1: Hook — "Here's how to [outcome] in [timeframe/steps]. A thread:"
Tweet 2: Step 1 — [WHAT + HOW, specific]
Tweet 3: Step 2 — [ends with a banger]
...
Tweet N: Summary + CTA ("Follow @handle for one thread like this a week.")
```

### Listicle Thread
Best for: authority content, saves, shares

```
Tweet 1: "[N] [tools/frameworks/lessons] for [outcome]. A thread:"
Tweet 2: "1/ [Item name] — [explanation with HOW, ends with banger]"
Tweet 3: "2/ [Item]..."
...
Final: "If this helped, follow @handle. New thread every [cadence]."
```

### Story Arc Thread
Best for: viral storytelling, case studies, personal journey

```
Tweet 1: Hook — the end state (the surprising outcome first)
Tweet 2–3: Context — where it started, what was tried
Tweet 4–6: The turning point — what actually changed things
Tweet 7: The lesson extracted
Final: CTA — "Follow for more breakdowns like this."
```

### Hot Take Thread
Best for: debate, differentiation, strong POV building

```
Tweet 1: Controversial claim — clear and unhedged
Tweet 2: The strongest objection + your rebuttal
Tweet 3–5: Evidence, examples, specifics
Tweet 6: The broader principle or implication
Final: "Agree? Disagree? Tell me below."
```

### Research / Data Thread
Best for: credibility, shareability, journalist/expert appeal

```
Tweet 1: Hook — the surprising finding
Tweet 2: Source and context
Tweet 3–5: Key data points, each with implication
Tweet 6: What this means in practice
Final: "Save this for your next [planning / pitch / conversation]."
```

## Engagement-Boosting Additions

These elements can be layered into any format:

- **Tagging**: Tag 4–6 relevant accounts per thread when genuinely applicable
- **Polls**: Add a poll to question posts to drive easy participation
- **Image**: A visual (chart, code screenshot, diagram) in the first tweet doubles impressions
- **Reply to yourself**: Add context or a bonus tip as a reply to your own top tweet
- **Reopen after 24h**: Reply to your own thread with "Still reading this? Here's the one thing people miss:" to revive reach

## CTAs That Work

Ranked by friction (low to high):

1. "Save this."
2. "Bookmark this thread."
3. "Follow @handle for more."
4. "What's your take? Drop it below."
5. "Share this with someone who needs it."
6. "Reply with your [question / example / result]."
7. "Join [newsletter] for the full system: [link]"

Match CTA friction to relationship warmth — cold audience gets low-friction asks first.
