# Code Slides

Create and evolve slide decks implemented in code, especially HTML/CSS/JS or React/TypeScript presentations, and generate AI-image slide packs from markdown or JSON content specs.

## Use this for

- new code-based slide decks
- redesigning or refactoring existing decks
- HTML presentation systems with responsive behavior
- PPT-to-web conversion and remote-control slide navigation
- presentation workflows that need code, assets, and styling to live together

## Install From The Monorepo

```bash
npx -y skills add ./marketing/skills/code-slides
mkdir -p ~/.codex/skills
cp -R marketing/skills/code-slides ~/.codex/skills/
```

Codex `$skill-installer` monorepo path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/marketing/skills/code-slides
```
