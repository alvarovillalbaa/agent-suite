# Composition Patterns

Use these patterns when a coded visual is the clearest representation.

## Product Dashboard Hero

- Strong frame or window chrome
- One focal KPI cluster
- One or two secondary panels
- Minimal but believable navigation and controls

## Terminal / Code Card

- Tight color palette
- Short code or command excerpt
- One key highlight rather than a full editor recreation
- Clear contrast and readable line spacing

## Device / Browser Frame

- Treat the frame as support, not the story
- Keep shadows and reflections subtle
- Make sure the interior canvas still reads without the frame

## Mixed Composition

- Generated or photographic background for mood
- Coded foreground for the product
- Annotation layer for the message

Use mixed composition when the product alone feels too clinical or the generated background alone feels too generic.
