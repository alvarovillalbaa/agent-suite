# References

This skill may use brand voice, style guides, or channel-specific format guidelines.
If a `references/` folder exists, use those files for tone, structure, and platform-specific patterns per channel.

## Available Reference Files

- **[title-formulas.md](./references/title-formulas.md)** — Proven headline formulas, power word categories, CTR optimization patterns, and title length guidelines by platform. Use when writing SEO-optimized titles.

- **[content-structure-templates.md](./references/content-structure-templates.md)** — Detailed markdown templates for blog posts, comparison articles, listicles, how-to guides, product reviews, pillar pages, and FAQ pages. Use when the user needs a specific content type.

Use during GEO Optimize mode.

- **[ai-citation-patterns.md](./references/ai-citation-patterns.md)** — How Google AI Overviews, ChatGPT, Perplexity, and Claude each select and cite sources. Includes optimal content structures for citation, citation likelihood factors, and per-engine priority items. Use to target specific AI engines.

- **[gap-analysis-frameworks.md](./references/gap-analysis-frameworks.md)** — Systematic keyword gap methodology (5-step), content format audit (with SERP feature alignment), funnel stage gap identification, multi-factor opportunity scoring model, and gap-to-calendar workflow with post-publication validation. Use during Content Gap Analysis mode.

Use when producing gap analysis deliverables.

Use as a reference model when running Content Gap Analysis mode.

- **[keyword-intent-taxonomy.md](./references/keyword-intent-taxonomy.md)** — Complete intent classification system with 4 primary types (Informational, Navigational, Commercial, Transactional), sub-categories, signal words, SERP feature correlations, mixed intent patterns, funnel stage alignment, and conversion potential by intent type. - **[keyword-research-example.md](./references/keyword-research-example.md)** — Complete example keyword research report for a project management software company targeting small businesses, showing executive summary, quick wins, growth keywords, GEO opportunities, topic clusters, and a content calendar. Use as a reference model when running Keyword Research mode.
