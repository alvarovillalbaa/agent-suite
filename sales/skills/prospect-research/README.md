# Prospect Research

Research skill for turning heterogeneous account and persona inputs into a decision-ready GTM brief.

## Use this for

- prospect and account research before outreach
- applying ICP, ICA, and signal scorecards
- building a brief that covers account, persona, and next best motion
- grounding outreach in current, attributable evidence

## Install

```bash
npx -y skills add ./sales/skills/prospect-research
mkdir -p ~/.codex/skills
cp -R sales/skills/prospect-research ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/sales/skills/prospect-research
```
