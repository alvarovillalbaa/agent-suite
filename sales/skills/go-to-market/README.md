# Go To Market

GTM skill for commercial planning, growth mechanics, messaging, and revenue-oriented execution.

## Use this for

- GTM planning and prioritization
- growth-engine thinking and commercial diagnostics
- packaging research into sales or growth action
- practical execution support across marketing, sales, and revenue operations

## Install

```bash
npx -y skills add ./sales/skills/go-to-market
mkdir -p ~/.codex/skills
cp -R sales/skills/go-to-market ~/.codex/skills/
```

Codex `$skill-installer` path:

```text
https://github.com/alvarovillalbaa/plugins/tree/main/sales/skills/go-to-market
```
