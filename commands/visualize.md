Use skill: **visualizer** — `skills/visualizer/SKILL.md`.

1. **Gather the source and goal** – Ask for the material, audience, artifact type, and output preference if those are missing.
2. **Choose the mode** – Select explainer, review, comparison, dashboard, timeline, or slide-mode.
3. **Generate the artifact** – Produce a self-contained HTML page that matches the chosen mode and audience.
4. **Keep it shareable** – Favor strong structure, clear takeaways, and a deliberate visual direction.
5. **Deliver** – Write the file, return the path, and summarize what the artifact communicates.
