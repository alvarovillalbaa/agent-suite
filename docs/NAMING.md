# Naming Conventions

These rules keep skills and agents consistent and discoverable.

## Skills

- Prefix required: `hreng-`
- Verbs are short: `hire-`, `perf-`, `org-`, `onboard-`
- One job per skill (no overlapping responsibilities)
- Avoid synonyms across skills (pick one term and stick to it)

## Agents

- Prefix required: `hreng-`
- Use domain nouns: `hreng-hiring`, `hreng-performance`, `hreng-org`
- Agents orchestrate skills and assemble templates; they do not replace skills
