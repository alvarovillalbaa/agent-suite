# Quickstart

## Install

```bash
claude --plugin-dir .
```

## Configure

```bash
cp templates/settings-template.md .claude/clous-hreng.local.md
```

## Try It

```bash
/clous:intake
/clous:onboard
/clous:health-check
```

## Orchestrator Agents

- `hreng-hiring`: intake → strategy → interview loop → evaluation
- `hreng-performance`: diagnosis → feedback/PIP plan
- `hreng-org`: skills inventory → ladder alignment → headcount plan
